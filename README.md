# 페어매칭관리 애플리케이션

미션을 함께할 페어를 관리해주는 애플리케이션입니다.

## 규칙

- 과정 : 백엔드 / 프론트 엔드
- 레벨 : 1,2,3,4,5
- 페어 매칭 조건
  - 미션을 함께 수행할 페어 두명씩 매칭
  - 홀수인 경우 한 페어는 3인으로 구성
  - 같은 레벨에서 이미 페어를 맺은 크루와는 다시 페어로 매칭 될수 없음
- 매칭 3회 시도까지 매칭이 되지 않거나 매칭을 할 수 있는 경우의 수가 없으면 에러 메시지를 출력한다.
- 에러 발생시 해당 부분부터 다시 입력받는다.

## 프로그램 흐름

1. 사용할 기능을 선택한다.
2. 해당 기능을 수행한다.
   - 1. 페어 매칭 (과정, 레벨, 미션 입력)
   - 2. 페어 조회 (과정, 레벨, 미션 입력)
   - 3. 페어 초기화
3. 기능 수행 후 다시 위 단계들을 반복한다.
4. Q를 입력하면 프로그램을 종료한다.

## 기능 목록

### 입력

- [x] 기능 번호를 입력한다.
- [x] 과정, 레벨, 미션을 입력한다.
- [ ] 예 | 아니오 답변을 입력한다.

### 입력값 검증

- [ ] 존재하는 기능 번호를 입력했는지 확인한다.
- [ ] 존재하는 과정을 입력했는지 확인한다.
- [ ] 레벨1 - 레벨5 사이를 입력했는지 확인한다.
- [ ] 해당 레벨의 미션을 입력했는지 확인한다.

### 페어 매칭

- [ ] 크루 목록의 순서를 랜덤으로 섞는다.
- [ ] 앞에서 부터 2명씩 매칭을 시작한다.
- [ ] 같은 레벨에서 페어로 만난적이 있는 크루인지 확인한다.
- [ ] 다시 매칭을 시도한다.

### 페어 조회

- [ ] 매칭된 페어 결과를 조회한다.

### 페어 초기화

- [ ] 저장된 미션에 대한 모든 페어 정보를 초기화한다.

### 출력

- [x] 과정,레벨,미션 정보를 출력한다.
- [ ] 페어 매칭 결과를 출력한다.
- [ ] 기능 수행 후 성공 결과를 출력한다.
